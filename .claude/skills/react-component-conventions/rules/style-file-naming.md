## ファイル名は kebab-case で統一する

すべてのファイルは kebab-case で命名する。

| 種別 | パターン | 例 |
|------|----------|----|
| コンポーネント | `kebab-case.tsx` | `user-profile.tsx` |
| hooks | `use-xxxx.ts` | `use-auth.ts` |
| ユーティリティ / ヘルパー | `kebab-case.ts` | `format-date.ts` |
| 型定義 | `kebab-case.ts` | `user-types.ts` |
| テスト | `kebab-case.test.tsx` | `user-profile.test.tsx` |
