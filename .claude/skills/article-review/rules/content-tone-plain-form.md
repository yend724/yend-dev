## 文体は「である調(常体)」を基本とし、「ですます調(敬体)」を使わない

記事本文は「である調 / 常体」で書く。「ですます調 / 敬体」は基本的に使わない。語尾は `〜である`、`〜だ`、`〜する`、体言止めなどを使い、`〜です`、`〜ます`、`〜でしょう` などは避ける。

**Bad(ですます調 / 混在):**

```md
この関数は引数を受け取って結果を返します。

返り値の型は処理内容に応じて決まります。
```

```md
この関数は引数を受け取って結果を返す。

返り値の型は処理内容に応じて決まります。  ← 混在
```

**Good(である調で統一):**

```md
この関数は引数を受け取って結果を返す。

返り値の型は処理内容に応じて決まる。
```

**チェック観点:**

- 段落ごとに語尾を確認する(`〜です` / `〜ます` / `〜でしょう` 系が混入していないか)
- 体言止めや `〜だ` / `〜である` / `〜する` の使い分けが自然か
- 引用ブロック内の文体は元の文体を尊重する(記事本文と異なってよい)
- コード中のコメントや出力例の文体は本ルールの対象外
